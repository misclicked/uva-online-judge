這題是求 A + A^2 + ... + A^k，A 為方陣。

一開始以為對於每個 A^i 用 O(lg i) 時間求矩陣再加總即可。但這樣還是會 TLE。

事實上矩陣的幾何級數是可以二分的，當 k 為偶數，則原式可二分為

> = [A + A^2 + ... + A^(K/2)] + [A^(K/2+1) + A^(K/2+2) + ... + A^(K/2+K/2)]
> = [A + A^2 + ... + A^(K/2)] + A^(K/2) [A^(K/2+1) + A^(K/2+2) + ... + A^(K/2+K/2)]
> = [1 + A^(k/2)] * [A + A^2 + ... + A^(K/2)]
